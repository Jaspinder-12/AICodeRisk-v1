# AICodeRisk Project Contract (V1)

This document establishes the legal and operational framework for the **AICodeRisk** project.

## 1. Project Purpose & Scope (V1)
The **AICodeRisk** project is a self-correcting and risk-aware AI coding environment.
- **Product Name**: AICodeRisk
- **Version**: V1
- **Primary Scope**: Analyze AI-generated **Python code** for **security vulnerabilities only**.
- **Output Format**: Strict **JSON risk report**.
- **Expansion Policy**: No expansion of scope or features is allowed until the public release.

## 2. Ownership and Authority
- **Exclusive Ownership**: All intellectual property, source code, documentation, and assets created within this project are the sole property of **Jaspinder S.** (the "Owner").
- **Final Decision Authority**: The Owner retains absolute authority over all code commits, architectural decisions, and project directions.

## 3. AI-Developer Interface (ADI)
- **Role of the AI**: The AI functions as an assistant and agent under the direct supervision of the Owner.
- **Verification**: All AI-generated code must be subjected to the project's risk-assessment protocols before being merged into the main branch.
- **Transparency**: The AI must provide clear rationales for its code modifications, particularly those involving "Self-Correction" logic.

## 4. Risk Mitigation Protocols
- **Risk Identification**: Every major code change must be evaluated for **Security Risk** in Python code.
- **Automated Guardrails**: The project will implement automated tests and linters to enforce safe coding practices and validate the strict JSON output format.

## 5. Liability and Disclaimer
- **Experimental Nature**: This project is highly experimental. The Owner and the AI assistant provide no guarantees regarding the stability or security of the generated code.
- **No Warranty**: All software is provided "as-is," without warranty of any kind.
